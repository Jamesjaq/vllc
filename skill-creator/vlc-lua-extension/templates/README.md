# VLC Infinity Example Extensions

This directory contains two complete examples of VLC Lua extensions:

1. **vlc-infinity.lua**: A basic extension that aggregates IPTV channels and public-domain movies.
2. **vlc-infinity-enhanced.lua**: An advanced extension that integrates TMDB metadata and multiple streaming providers for a complete movie browsing experience.

Use these files as references or starting points when building new VLC Lua extensions.
